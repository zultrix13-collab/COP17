### 3. ✅ PrivacyInfo.xcprivacy ↔ Privacy Policy зөрчил (ЗАСАГДСАН)
- `apps/mobile/ios/Runner/PrivacyInfo.xcprivacy` нь **DeviceID (linked)** болон **PreciseLocation** цуглуулдаг гэж зарласан байна.
- Харин App Store Connect-ийн "App Privacy" мэдээлэл нь зөвхөн **Coarse Location** болон User ID/Email/Name-ийг зарласан байсан — Device ID-г дурдаагүй, location нь "precise" эсэхийг тодорхойлоогоогүй байсан.

**Шийдэл (2026-06-13):** App Store Connect → App Privacy дээр "Edit Data Types" ашиглан:
- Location: "Coarse Location"-г хассан, "Precise Location" нэмж, App Functionality зориулалттай, хэрэглэгчийн identity-д **холбоогдоогүй**, tracking-д ашиглагдахгүй гэж тохируулсан.
- Identifiers: "Device ID" нэмж, App Functionality зориулалттай, хэрэглэгчийн identity-д **холбоогдсон** (FCM push token нь `/device-tokens`-ээр хэрэглэгчийн account-д холбогддог тул), tracking-д ашиглагдахгүй гэж тохируулсан.
- Publish хийсэн. Эцсийн ASC App Privacy: **5 data types — Device ID, User ID, Precise Location, Name, Email Address**, бүгд App Functionality зориулалттай; Device ID/User ID/Name/Email — Linked to identity; Precise Location — Not linked. Энэ нь `PrivacyInfo.xcprivacy`-тэй бүрэн нийцэж байна.
